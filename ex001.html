Exercícios Python - EX005

IMC

Este programa calcula o Índice de Massa Corporal (IMC) e fornece uma avaliação da condição de saúde com base no resultado.

Dicas sobre o código

  • Cálculo do IMC: O IMC é calculado dividindo o peso (em kg) pela altura ao quadrado (em metros).
  • Estrutura condicional: O programa usa uma série de condições if/elif/else para classificar o IMC em diferentes categorias.
  • Strings multilinha: Utilizamos aspas triplas para criar strings que ocupam várias linhas, mantendo a formatação.
  • Formatação de números: O f-string {imc:.2f} exibe o IMC com 2 casas decimais.
  • Conversão de tipos: Convertemos a entrada do usuário para float usando float(input()) para permitir cálculos matemáticos.
  • Saída formatada: O resultado final combina o nome do usuário, o valor do IMC e a condição de saúde em uma mensagem formatada.
