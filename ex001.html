EX005 - Manipulação de Listas

Iniciante Listas

Crie um programa que manipule uma lista de números, realizando operações de adição, remoção e modificação de elementos.

Saída:

    

    
    

        
Dicas:

        
    
            
  • Use o método append() para adicionar elementos ao final da lista
    • 
            
  • Use o método remove() ou pop() para remover elementos
    • 
            
  • Você pode modificar um elemento específico usando seu índice: lista[índice] = novo_valor
    • 
            
  • Lembre-se que os índices em Python começam em 0
    • 
            
  • Use print() para exibir a lista modificada
